### **Naming Convention:**
```
Format: ##-Capitalized-Words-With-Hyphens.md

Examples:
‚úÖ 01-Learning-Roadmap.md
‚úÖ 02-Java-Resources.md
‚úÖ 03-Selenium-Resources.md

```

### **Numbering Rules:**
- **Sequential**: Start at 01, no gaps
- **Leading Zero**: Always use 01, 02... not 1, 2
- **Directory Scope**: Numbering restarts in each directory
- **Logical Order**: Follow learning/implementation progression

### **Content Rules:**
- **File Names**: Describe content clearly
- **No Abbreviations**: Use full words for clarity
- **Consistent Casing**: Title Case for all words
- **Hyphen Separators**: Use hyphens, not underscores or spaces
